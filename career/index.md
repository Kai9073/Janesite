# 高中選科資訊

### 《高中選科及升學指南 2022》
為協助中三學生規劃高中選科，以及了解其他出路，學友社出版《高中選科及升學指南2022》，提供高中選科策略、最新大專院校收生要求、高中以外出路等升學資訊等。

指南涵蓋中三同學面對選科及升學時所需的實用資料，包括：
- 高中課程、優化措施及DSE最新資訊
- 高中選科策略及聯招大學收生負責人的選科意見
- 最新本地大專院校的收生資訊
- 聯招課程與高中科目關係一覽表
- 高中以外全日制課程、學徒及見習訓練計劃
